# Smart Diag Backend - Symfony + Docker

Backend API module for diagnostics and Ford VIN decoding.

## Local Features

- normalizes VIN values,
- validates VIN format,
- calculates the check digit,
- decodes Ford WMI values,
- decodes the model year,
- returns the plant code,
- returns the serial number,
- optionally queries the NHTSA vPIC API.

## Running

```bash
docker compose up --build
```

Database migrations:

```bash
make migrate
```

The API will be available at:

```text
http://localhost:8000
```

Healthcheck:

```bash
curl http://localhost:8000/health
```

Swagger UI:

```text
http://localhost:8000/docs
```

The documentation is generated by NelmioApiDocBundle from Symfony routes and OpenAPI metadata.

Redis browser:

```text
http://localhost:8081
```

OpenAPI specification:

```text
http://localhost:8000/docs/openapi.json
```

Local decoding:

```bash
curl -H "X-License-Key: <license>" http://localhost:8000/api/vin/ford/1FA6P8CF0F5300000
```

Decoding with external NHTSA enrichment:

```bash
curl -H "X-License-Key: <license>" "http://localhost:8000/api/vin/ford/1FA6P8CF0F5300000?external=1"
```

Create a license:

```bash
curl -X POST http://localhost:8000/api/admin/licenses \
  -H "X-Admin-Key: docker-admin-secret" \
  -H "Content-Type: application/json" \
  -d '{"clientName":"ACME","clientEmail":"admin@example.com","note":"test","validUntil":"2027-06-21T23:59:59+00:00"}'
```

## Quality Checks

```bash
make analyse
make cs
make check
```

- `make analyse` runs PHPStan.
- `make cs` runs PHP_CodeSniffer with the PSR-12 ruleset.
- `make check` runs PHPUnit, PHPStan, and PHPCS through Composer.

## Important

Full decoding of VIN positions 4-8 depends on the market, model year, model, and manufacturer data. This project provides a safe local foundation and leaves room to extend it with custom Ford lookup tables.
